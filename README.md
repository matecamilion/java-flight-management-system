# Java Flight Management System

Console-based application developed in Java for managing flights, users, and reservations.

## Features
- User management system  
- Flight and reservation handling  
- Payment methods implementation  
- Object-oriented design with multiple classes  

## Tech Stack
- Java  
- Object-Oriented Programming (OOP)  

## Description
This project was developed as part of my academic training and focuses on applying object-oriented programming principles such as encapsulation, modular design, and business logic implementation.
